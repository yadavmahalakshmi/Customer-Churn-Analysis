# Customer Churn Analysis
End-to-end Customer Churn Analysis using Python, SQL, and Power BI with actionable business insights and interactive dashboards.

**Author: Mahalakshmi Yadav**

## Project Overview
This project focuses on analyzing customer churn in the telecom industry to identify key factors influencing customer retention and provide data-driven business recommendations.

## Objectives
- Identify customers likely to churn  
- Analyze key factors influencing churn  
- Provide actionable insights to improve customer retention  

## Tools & Technologies
- **Python** – Data cleaning and preprocessing  
- **SQL** – Data querying and transformation  
- **Power BI** – Interactive dashboard and visualization  

## Key Highlights
- End-to-end data analysis project  
- Interactive Power BI dashboard  
- Real-world telecom dataset  
- Business-focused insights and recommendations

## Dataset
- Contains ~7,000+ customer records  
- Includes:
  - Demographics (Gender, Senior Citizen)
  - Services (Internet, Phone Service)
  - Account Details (Contract Type, Payment Method)
  - Financial Data (Monthly Charges, Total Charges)
  - Target Variable: Churn (Yes/No) or (1/0)

## Data Cleaning & Preparation
- Converted data types (Total Charges to numeric)  
- Handled missing and null values  
- Removed inconsistencies  
- Standardized categorical variables  
- Ensured data quality for analysis

## Dashboard & Analysis

### Key Insights
- Month-to-month customers have the highest churn  
- Fiber optic users show higher churn rates  
- Higher monthly charges increase churn probability  
- Long-term contracts reduce churn significantly  

## Business Recommendations
- Promote long-term contracts to improve retention  
- Improve service quality for high-risk customers  
- Offer personalized plans and discounts  
- Monitor customers with high monthly charges  

## Future Scope
- Build Machine Learning models for churn prediction  
- Develop real-time dashboards
